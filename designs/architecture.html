Mneme — System Architecture

Edge-first memory for AI agents · v0.1 MVP

Transport

  • HTTP REST API on localhost:8989
  • MCP protocol via stdio / SSE
  • CLI: mneme add/search/serve
  • Any MCP client connects zero config

Engine

  • Store: write + vectorize + classify
  • Search: semantic + BM25 + filter
  • Quality (V2): contradiction + decay
  • Types: fact / preference / event / conversation / skill

Storage

  • SQLite single-file database
  • sqlite-vec for vector search
  • ~4ms p99 at 100k vectors (384d)
  • 45MB memory for 100k entries
