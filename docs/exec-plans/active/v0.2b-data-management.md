## 本次实施范围（MVP）

### 1️⃣ 翻页列表 `GET /v1/memories`
```
GET /v1/memories?offset=0&limit=20&type=fact&tags=python
     &date_from=2026-01-01&date_to=2026-06-15
     &weight_min=0.5&weight_max=2.0
     &sort_by=created_at&sort_order=desc
     &include_deleted=false

→ { "results": [...], "total": 150, "offset": 0, "limit": 20 }
```

### 2️⃣ 批量操作
```
POST /v1/memories/batch/delete   → { filters: { type, tags, date_from, date_to, weight_min } }
POST /v1/memories/batch/update   → { filters: {...}, updates: { tags, weight, type } }
POST /v1/memories/batch/restore  → 恢复软删除的记忆
```

### 3️⃣ 导出/导入
```
GET  /v1/export?format=json  → JSON 文件下载
POST /v1/import              → JSON 文件上传导入
```

### 4️⃣ CLI 命令增强
```bash
mneme list [--type] [--tags] [--limit] [--offset]   # 翻页列表
mneme export --output memories.json                  # 导出
mneme import memories.json                           # 导入
mneme stats --detail                                 # 详细统计
mneme delete --type fact --tags temp                 # 批量删除
mneme restore <memory_id>                            # 恢复
```

### 5️⃣ 增强统计
```bash
GET /v1/stats/detailed
→ { total, by_type, by_tag, total_weight, memory_growth: [...], deleted_count }
```
