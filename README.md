# AI Memory System — 纯离线 Agent 记忆系统

## 定位

纯离线、即插即用的 AI Agent 记忆系统。主动适配各种 Agent 框架，留出强扩展性接口。

## 核心差异化

1. **端侧优先（Edge-First）** — 离线可用，在线可选
2. **记忆质量管理** — 矛盾检测、过期管理、修正学习
3. **即插即用** — HTTP API + MCP 协议，任何 Agent 都能接
